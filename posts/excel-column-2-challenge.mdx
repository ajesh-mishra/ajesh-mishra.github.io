## Solution

```rust
fn title_to_number(column_title: String) -> i32 {
  let mut result = 0;
  for (i, c) in column_title.chars().rev().enumerate() {
    let pos = c as i32 - 64;
    result +=  26_i32.pow(i as u32) * pos;
  }
  result
}
```

## Main Function

fn main() {
  let result = title_to_number("FXSHRXW".to_string());
  println!("2147483647\n{}", result);
}
