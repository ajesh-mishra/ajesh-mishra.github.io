## Solution

```rust
fn length_of_last_word(s: String) -> i32 {
  let mut count:i32 = 0;
  
  for c in s.chars().rev() {
    if count == 0 && c == ' ' {
      continue;
    } else if c == ' ' {
      break;
    } else {
      count += 1;
    }
  }
  
  count
}
```

## Main Function

```rust
fn main() {
  let input = String::from("hello world");
  let result = length_of_last_word(input);
  
  println!("Length of Last word is {}", result);
}
```
