## Solution

A phrase is a palindrome if, after converting all uppercase letters into lowercase letters and removing all non-alphanumeric characters, it reads the same forward and backward. Alphanumeric characters include letters and numbers.

```rust
fn is_palindrome(s: String) -> bool {
  let mut given_str = String::from("");
  let mut rev_str = String::from("");

  for c in s.chars() {
    match c {
      'a'..='z' | 'A'..='Z' | '0'..='9' => {
        given_str = format!("{}{}", given_str, c.to_lowercase().to_string());
      },
      _ => {},
    };
  }

  for c in given_str.chars().rev() {
    rev_str.push(c);
  }

  if given_str == rev_str {
    return true;
  }

  false     
}
```

## Main Function

```rust
fn main() {
  let input = String::from("A man, a plan, a canal: Panama");
  let result = is_palindrome(input);
  println!("{}", result);
}
```

```rust
#[cfg(test)]
mod tests {
  use super::*;
  
  #[test]
  fn tc_1() {
    let input = String::from("A man, a plan, a canal: Panama");
    assert_eq!(is_palindrome(input), true);
  }

  #[test]
  fn tc_2() {
    let input = String::from("race a car");
    assert_eq!(is_palindrome(input), false);
  }

  #[test]
  fn tc_3() {
    let input = String::from(" ");
    assert_eq!(is_palindrome(input), true);
  }
}
```
